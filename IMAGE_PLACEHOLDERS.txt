HOW TO ADD IMAGES
=================
  Static images (hero, contact):
    1. Place the file in the static/images/ directory with the exact filename shown above.
    2. Run: python manage.py collectstatic (in production) or just use DEBUG=True (dev).
    3. The image will appear automatically — no code changes needed.

  Project images (via admin):
    1. Run the server: python manage.py runserver
    2. Log in at http://127.0.0.1:8000/admin/
    3. Navigate to Portfolio > Projects > [project name]
    4. Use the "image" field to upload the screenshot.
    5. Save — the image will immediately appear on the portfolio.
